Course taught by Danny Thorne.

We are building this game using the free version of unity and with a repository on GitHub. The game ideas and concepts can be viewed on the github repository.

Our inspiration for the game is a Metroid style exploration and combat with Portal style puzzle solving.

Each team member was assigned certain developments of the game. They are:

  • Jeff Hartman: Enemy AI
  • Isaac Meisner: Player Controls
  • TJ Broadus: Gun Controls
  • Laura Buckman: Scene Design
  • Danny Thorne: Futher help on any developments
