Space-Adventure (name not final) is a Super-Metroid style 2-d sidescroller with an emphasis on puzzle solving. We are building this game using the free version of Unity. Source code and other bits about the game can be viewed on github. Feel free to fork and contribute! Each team member was assigned certain developments of the game. They are:

  • Jeff Hartman: Enemy AI
  • Isaac Meisner: Player Controls
  • TJ Broadus: Gun Controls
  • Laura Buckman: Scene Design
  • Danny Thorne: Futher help on any developments
